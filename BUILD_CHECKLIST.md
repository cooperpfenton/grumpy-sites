# Grumpy Sites — Website Build Checklist

Use this for every client website. Do not skip any step.

## Before Starting
- [ ] Signed contract with client
- [ ] 50% deposit received
- [ ] Client provided: business name, text content, images, logo (if they have one)
- [ ] Scope of work defined in writing (pages, features, revision limit)

## During Build
- [ ] All fonts are Google Fonts or system fonts (no paid licenses)
- [ ] All images are owned by client, taken by you, or from free stock (Unsplash/Pexels)
- [ ] No fake reviews or testimonials
- [ ] All images compressed for web (under 300 KB each)
- [ ] Images have width/height or aspect-ratio set (prevents layout shift)
- [ ] No hardcoded passwords or secret API keys in client-facing code
- [ ] All external links use HTTPS
- [ ] Contact form tested and working
- [ ] Form validation: all required fields show error if empty
- [ ] Mobile responsive (test at 375px, 768px, 1024px)
- [ ] Favicon added
- [ ] Meta description on every page
- [ ] Privacy policy page included if site collects any data
- [ ] Copyright year in footer is correct

## Design Standards
- [ ] Use Google Fonts (Poppins + Open Sans as default, or match client's brand)
- [ ] SVG icons only (Lucide, Heroicons) — never emojis as structural icons
- [ ] 8px spacing rhythm
- [ ] Minimum 16px body text
- [ ] Minimum 4.5:1 color contrast ratio for text
- [ ] Visible focus states for keyboard navigation
- [ ] prefers-reduced-motion respected
- [ ] Touch targets minimum 44x44px

## Before Delivery
- [ ] Tested on Chrome, Safari, and Firefox
- [ ] Tested on actual phone (not just browser resize)
- [ ] All links work (no broken links)
- [ ] Contact form sends to correct email/number
- [ ] Page load time under 3 seconds
- [ ] No console errors
- [ ] Privacy policy page linked in footer
- [ ] Client logins documented (hosting, domain, form service)

## Handoff
- [ ] Remaining 50% payment received
- [ ] Domain and hosting in client's name
- [ ] Client given all login credentials
- [ ] Simple handoff doc: how to update basic content, who to contact for help
- [ ] Site added to Grumpy Sites portfolio (with client permission)
- [ ] Keep a backup copy of the site files
